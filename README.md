# Try it yourself
First install loop:
```
$ npm install --global devloop
```
Then clone the repo and run the loop build process:
```
$ git clone git@github.com:jqcoffey/qwebmon.git
$ cd qwebmon
$ loop
=> Now browse to http://localhost:8080
```
